# ScreenBlocker

一个给竖屏副显示器使用的 macOS 菜单栏工具。

它会在竖屏上方生成一块遮挡区，并尽量把其他应用窗口约束到下半部分，让内容停留在更自然的视线范围内，减少频繁抬头带来的颈部不适。

## 适合谁

- 有一块竖着放的副显示器
- 觉得副屏太高，只想使用下半部分
- 长时间使用副屏时，频繁抬头会让脖子容易发紧或不舒服
- 想让常用窗口尽量贴着下半区工作

## 功能

- 在竖屏上方显示遮挡层
- 自动把普通窗口压回可用区域
- 尽量处理窗口全屏或铺满后的回退位置
- 保留顶部菜单栏区域，不遮住系统菜单栏
- 提供菜单栏入口和设置面板

## 使用要求

- macOS 14 或更高版本
- 至少一块竖屏显示器
- 需要开启“辅助功能”权限，否则应用无法移动其他窗口

## 安装

从 [GitHub Releases](https://github.com/luckfunc/ScreenBlocker/releases) 下载 `.dmg` 安装包即可。

## 使用方式

- 启动后从菜单栏打开设置
- 调整上方遮挡比例
- 给应用授予“辅助功能”权限
- 把窗口拖到竖屏时，应用会尽量把它们压回可用区域

## 注意

- 一些应用的特殊窗口、全屏模式或自绘窗口不一定完全受控
- 多显示器布局非常特殊时，仍然可能需要继续微调窗口吸附逻辑
- 当前安装包可能没有开发者签名，首次打开时 macOS 可能会提示确认
